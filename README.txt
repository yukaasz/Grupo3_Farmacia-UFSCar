ROTEIRO DE EXECUÇÃO DA APLICAÇÃO 

1) Crie o banco de dados no PostgreSQL com os scripts fornecidos (tabelas, triggers, dados).
2)Configurar no arquivo “server.js”, das linhas 14 a 20, as credenciais do seu banco de dados local.
3) Instalar o node.jv, presente neste link.
4) Como é um script não assinado, pode dar um erro no npm, utilize esse comando no terminal:
5) “Set-ExecutionPolicy RemoteSigned -Scope CurrentUser”
6) Utilize “npm install” caso esteja faltando algum componente
7) Utilize “node server.js” para abrir o servidor
8) Abra o arquivo “login.html” para iniciar a aplicação
9) Para efetuar o login, utilize a credencial já autorizada pela aplicação a seguir:
    a) CPF: 12345678901
    b) SENHA: 12345
